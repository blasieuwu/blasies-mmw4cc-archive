# blasieuwu's MMW4CC archive

An archive made by **@blasieuwu** that contains:
1. The **original** MMW4CC .zip
2. An old modified version of MikuMikuWorld that contains a **score viewer** to see what your charts look like

## What is MMW4CC?

MMW4CC (also known as **MikuMikuWorld for Chart Cyanvas**) is a tool people use to make charts for Chart Cyanvas.

Chart Cyanvas is a server where people can host their custom **Project SEKAI: Colorful Stage!** charts.

You might be asking: "Why am I even archiving MikuMikuWord for Chart Cyanvas?"

Recently, on April 27, 2026, Colorful Pallete & SEGA (the creators of Project SEKAI: Colorful Stage!) recently announced that **any fanmade charts that use official assets** will be banned (you can find the original X (formerly Twitter) post [here](https://x.com/pj_sekai/status/2048729214900461995))

As a result, Nanashi, the creator of Chart Cyanvas has made her repositories private.

(you may find these quotes at their [original source](https://gist.github.com/sevenc-nanashi/5f7616eb9655cf6ec4ae84e9ed4d036b))

---

"As the official **officially** [stated to ban fanmade charts using official assets](https://x.com/pj_sekai/status/2048729214900461995) and they requested to delete existing videos, I've made **my repositories private, so these things became unavailable. This means you cannot download MMW4CC, play on Chart Cyanvas, and so on.**
- Chart Cyanvas Archive
- Potato Leaves
- MikuMikuWorld for Chart Cyanvas
- Chart Cyanvas Source Code
- Invites for Chart Cyanvas Discord server"

"I've deleted my works on 2026/05/04 JST, which is a day before they starts to officially ban users.

Thank you for playing on Chart Cyanvas, and my fanmade things."

---

This became the reason why this repository was created: It was made to archive the charting software.

Unfortunately, we, as the charting community, cannot do anything about this. **"Officials have the permissions to nuke everything, you know."**

**Disclaimer: This repository is a community-maintained archival mirror for historical preservation. It is completely unaffiliated with the original creator (@sevenc_nanashi) or Project SEKAI officials.**
